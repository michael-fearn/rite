# Install baseline VM collectors through VM Configure

Status: done

## What to build

Teach VM Configure to apply default VM-level Instrumentation to ordinary VMs by installing and configuring the baseline collector set. The first baseline is node exporter for system metrics and Grafana Alloy for VM log collection, skipped only when the VM explicitly opts out.

## Acceptance criteria

- [x] VM Configure installs and enables node exporter and Grafana Alloy for instrumented ordinary VMs.
- [x] VM Configure skips baseline collectors for VMs with VM-level Instrumentation disabled.
- [x] Generated collector configuration uses the declared VM identity and routes logs toward the Observability Service on the VM network.
- [x] Tests cover default-on behavior, explicit opt-out behavior, and idempotent reconfiguration.
- [x] The implementation keeps per-VM collector profile selection deferred.
