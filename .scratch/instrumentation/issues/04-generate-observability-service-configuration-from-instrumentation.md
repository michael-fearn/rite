Status: done

## What to build

## Acceptance criteria

- [x] Prometheus configuration includes scrape targets for instrumented VMs using declared static VM IP addresses.

- [x] Blackbox-style probe configuration includes Service `http_probe` Telemetry Targets using Backend VM IP and declared Published Port by default.
- [x] Loki ingestion endpoints expected by VM-local Alloy are represented in the generated observability configuration.
- [x] Golden or focused tests prove generated config for VM metrics, Service metrics, HTTP probes, opt-out VMs, and default scheme/path behavior.
